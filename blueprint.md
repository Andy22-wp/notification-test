# Project Overview

This project is a simple Angular application that demonstrates how to send a notification to a mobile device when a button is clicked.

# Features

*   A simple UI with a button.
*   Clicking the button triggers a local notification.

# Plan

1.  **Phase 1: Project Setup and Basic UI**
    *   Create a new Angular application.
    *   Create a main `app` component with a button.
2.  **Phase 2: Implement Notification Logic**
    *   Add a service to handle notifications.
    *   Implement the logic to request notification permission.
    *   Implement the logic to send a notification.
3.  **Phase 3: Connect UI and Logic**
    *   Connect the button click event to the notification service.
